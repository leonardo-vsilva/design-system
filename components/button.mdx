| Variante | Ênfase | Uso |
|----------|--------|-----|
| `default` | Alta | Ação principal — salvar, confirmar, criar |
| `secondary` | Média | Alternativa válida — salvar rascunho, duplicar |
| `destructive` | Alta | Ação irreversível — deletar, remover |
| `outline` | Baixa | Ação secundária visível — editar, exportar, cancelar |
| `ghost` | Mínima | Ação discreta — ícones em hover, contextos densos |
| `link` | Mínima | Ação que parece link de texto |

## Hierarquia

> **A ação de menor compromisso fica à esquerda. A ação de maior compromisso fica à direita.**

### Linhas de ação

Três regras para toolbars e cabeçalhos de seção:

**1 — Máximo 1 `default` por página.** Modais isolados podem ter seu próprio `default` sem conflitar com o da página.

**2 — Máximo 2 botões visíveis por linha.** Quando há 3 ou mais, agrupe os excedentes em um `DropdownMenu` com trigger `•••`. O `default` nunca vai para o dropdown.

```erb

<%= render RubyUI::Button.new(variant: :outline, size: :sm) { "↓ Exportar" } %>
<%= render RubyUI::DropdownMenu.new do %>
  <%= render RubyUI::DropdownMenuTrigger.new do %>
    <%= render RubyUI::Button.new(variant: :outline, size: :icon) do %>
      <%= lucide_icon "ellipsis", class: "size-4" %>
    <% end %>
  <% end %>
  <%= render RubyUI::DropdownMenuContent.new(align: "end") do %>
    <%= render RubyUI::DropdownMenuItem.new { "Histórico" } %>
    <%= render RubyUI::DropdownMenuSeparator.new %>
    <%= render RubyUI::DropdownMenuItem.new(class: "text-destructive focus:text-destructive") { "Excluir todos" } %>
  <% end %>
<% end %>
<%= render RubyUI::Button.new(variant: :default, size: :sm) { "+ Incluir novos" } %>
```

### Padrões por contexto

| Contexto | Ordenamento (esq → dir) |
|----------|------------------------|
| Modal de confirmação | `Outline: Cancelar` → `Default: Confirmar` |
| Modal destrutivo | `Outline: Cancelar` → `Destructive: Excluir` |
| Formulário com rascunho | `Outline: Cancelar` → `Secondary: Salvar rascunho` → `Default: Publicar` |
| Toolbar de listagem | `Outline: Exportar` · `⋯` → `Default: Novo item` |
| Ação em linha (tabela) | `Outline: Editar` · `Outline: Duplicar` |
